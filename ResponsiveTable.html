This page is an example of creating a responsive table. The source of the page contains everything you need. The page will "respond" when the browser width is <=600px. The header text will shift to the left of the data cells in responsive mode.

In order to make this work, I added a custom attribute to each TD tag called header-label. The value of header-value is the text you want to appear to the left of the data in reponsive mode. In this case the value of the header-label is the same as the value of the TH tags, but that doesn't have to be the case.

Of course, all this would be generated on the server side. That way, you can dynamically created your "header-label" if desired. Or if the column names are always the same for your need you can hard code the values in the CSS statically.
